1.O que é chamado de JOIN em um banco de dados?
Permite fazer uma consulta cruzando os dados de duas tabelas ou mais.

2. Nomeie e explique 2 tipos de JOIN.
INNER JOIN - Retorna uma interseção entre duas tabelas ou mais.
LEFT JOIN - Retorna a interseção e os dados da primeira tabela da consulta.

3. Para que é usado o GROUP BY?
Agrupa dados de uma coluna específica.

4. Para que é usado o HAVING?
Determina uma condição de busca ao usar o GROUP BY.
